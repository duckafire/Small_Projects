# stdbool
Adiciona 4 macros para explicitar o uso de valores booleanos, assim facilitando a leitura do código.

<hr>

| Macros | Valor |
| :-:    | :-:   |
| bool   | `int` |
| true   | `1`   |
| false  | `0`   |

<br>

> A macro `__bool_true_false_are_defined` também está inclusa neste cabeçalho.
