# stdbool
Adiciona 4 macros relacionados a valores booleanos (tipo e valores), permitindo a declaração explicita destes.

<br>

| Macros | Valor |
| :-:    | :-:   |
| bool   | Tipo booleano (`int`)  |
| true   | 1     |
| false  | 0     |
| \_\_bool\_true\_false\_are\_defined | 1 |
