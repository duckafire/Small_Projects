# assert
Fornece **uma** função que verifica expressões booleanas.

<br>

* void assert(int);
	* Comportamento: caso `int` seja igual a `0`, interrompe a execução do programa e imprime uma mensagem de erro, do contrário nada é feito.
	* Retorno: nenhum.
